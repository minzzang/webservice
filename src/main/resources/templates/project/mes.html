메인 프로젝트 상세정보

웹 MES 솔루션 개발

2019.07 ~ 2020.09

프로젝트 개요

제조업에 필요한 입고, 주문, 출고, 창고, 생산을 관리해주는 MES 프로젝트입니다.

관리자 화면과 작업자 화면이 있으며 저는 작업자 화면을 주로 개발했습니다.

작업자 화면은 현장에서 작업자들이 패널PC와 태블릿을 이용하여 입고, 생산, 출고를 할 수 있는 화면입니다.

업체가 요구하는 기능에 맞게 커스터 마이징했으며 현재 여러 업체에 서비스하고 있습니다.

배운 점

처음 작업자 화면을 맡았을 때 많은 기능이 이미 구현되어 있었습니다.

그런데 길고 긴 코드에는 주석이 하나도 없었고 메소드 하나에 너무 많은 기능이 있었습니다.

코드를 읽기도 너무 어려웠고 작은 기능을 변경하는 것도 어려웠습니다.

메소드 하나당 하나의 기능만 담당하게 리팩토링했고

변수나 메소드의 이름을 처음 봐도 알 수 있게 축약된 표현 대신 풀 네이밍으로 변경했습니다.

리팩토링을 진행하면서 어렵고 힘든 점도 많았지만, 코드에 대해 많이 생각하게 되고 객체지향에 관심을 두게 되었습니다.


업체의 새로운 요구사항을 반영하면서 기능을 수정했는데 기존의 저장 로직이 비효율적인 것을 깨달았습니다.

공정 리스트의 row마다 Ajax로 넘겨주어 row가 늘어날수록 시간이 오래 걸리는 문제점이 있었습니다.

전체 리스트를 Ajax로 넘겨주어 서버에서 하나씩 저장하는 로직으로 바꿔 시간을 단축했습니다.


아쉬운 점으로는 테스트 코드를 작성하지 않았던 것입니다.

테스트 코드가 있었다면 리팩토링할 때도 더 쉽게 코드를 수정할 수 있었을 것이고

새로운 기능을 추가했을 때 기존 코드를 수동으로 테스트하는 시간을 아낄 수 있었을 것입니다.
